
Purpose

To create a JavaScript Multiple Choice quiz using all of the things I have learned in the Full-Stack Coding Bootcamp so far.

How to take the quiz

- click on Begin Quiz to start the quiz
  - After Quiz start's you have to beat the timer
  - The questions are Multiple Choice pick the correct answer by clicking on an answer
  - If you pick the wrong answer "Incorrect" will appear"
  - If you pick the right answer  "Correct" will appear"

Site Makeup

- HTML (HyperText Markup Language)
- CSS (Cascading Style Sheets)
- JS (JavaScript)
